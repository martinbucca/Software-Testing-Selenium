# Software-Testing-Selenium
Software Testing using selenium with Java. Website tested: https://practicetestautomation.com/practice-test-login/


# Run with Docker

### Step 1:
``` 
docker compose up
```
### Step 2:
``` 
docker exec -it selenium-big-assignment-ubuntu-1 bash
```
### Step 3:
```cd tests```

### Step 4:
```gradle test```
